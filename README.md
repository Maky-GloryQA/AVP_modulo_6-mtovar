**Módulo 6: Inferencia Estadística | Alkemy**

## 📋 Descripción del proyecto

Este proyecto es la evaluación del módulo de Inferencia Estadística. El objetivo es realizar un análisis estadístico completo sobre los hábitos de sueño, alimentación y actividad física en jóvenes universitarios, siguiendo el método científico.

## 📊 Dataset

El dataset fue **simulado** con 150 registros de estudiantes universitarios de 18 a 28 años. Incluye 18 variables entre cuantitativas y cualitativas:

- Variables cuantitativas: edad, horas de sueño, horas de estudio, IMC, uso del celular, comidas por día, satisfacción de vida
- Variables cualitativas: género, carrera, nivel de estrés, actividad física, tipo de dieta, etc.

## ❓ Pregunta de investigación

> ¿Los estudiantes universitarios duermen en promedio menos de 7 horas por noche, y se relaciona esto con su nivel de estrés y actividad física?

## 🔬 Hipótesis

- **H₀:** µ ≥ 7 horas (los estudiantes duermen lo suficiente)
- **H₁:** µ < 7 horas (los estudiantes duermen menos de lo recomendado)
